# 一、产品定位

项目名称：

B2B SEO OS / B2B 独立站站内 SEO 运营工作台

产品本质：

这是一个只面向 **WordPress B2B 外贸独立站** 的本地站内 SEO 运营系统。

它不是普通 SEO 报表工具。
它不是 AI 写文章工具。
它不是 WordPress 发布器。
它不是多站点后台。
它不是 Shopify / Webflow / 自研站适配器。
它不是供应商筛选或供应商验证工具。
它不是 CRM、外链、广告或 EDM 系统。

它的核心流程是：

```text
接入 WordPress 网站
→ AI / Agent 读取网站现状
→ 审计现有网站
→ 建立 B2B 上下文与证据库
→ 生成种子词库
→ 人工上传多个关键词 CSV
→ 程序内置脚本合并去重
→ AI + 规则清洗
→ 写入总关键词数据库
→ 人工审核关键词
→ 分配适合现有页面的关键词
→ 根据审计结果修复页面
→ 不适合现有页面的词进入未使用词池
→ 对未使用词做超级聚类
→ 生成新页面 / 内容任务
→ Content Engine 执行内容生产
→ QA / 交付 / 持续监控
```

这次前端原型必须围绕这个主流程，不要回到旧版“先做关键词地图再审计”的新站建站逻辑。
